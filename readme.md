# μ(t) Official Webiste

Built with Next.js

## Running
Assuming you have [Node.js](https://nodejs.org/en), in your terminal, do `npm install` (install packages), then `npm run dev` (start the server). Your development server will be up, probably at http://localhost:3000.

## Adding Events
1. Create a new index.js file in `pages/events/EVENT_NAME/index.js`
2. Copy over the contents of `pages/events/example/index.js`
3. Supply an image, which by default is expected at `pages/events/EVENT_NAME/image.png`
